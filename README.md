# Mendeleev Periodic Table

This project is an educational platform for learning the periodic table of elements. It provides complete information about 118 chemical elements in a modern and interactive environment.

## Features

### 1. Main Interface (Desktop)
The table is designed to be clear and organized. In the actual application, all element cells have rounded corners and professional hover animations that highlight each element when you move your mouse over it.

![Main Page Desktop](https://raw.githubusercontent.com/adfchcjnkb/periodic-table/refs/heads/main/main%20page%20(Desktop)%20.png)

### 2. Atomic Model Visualization
For each element, an interactive atomic model (Bohr model) is displayed. It shows the nucleus and the correct number of electron shells with animated electrons.

![Atomic Model](https://raw.githubusercontent.com/adfchcjnkb/periodic-table/refs/heads/main/atomic%20model.png)

### 3. Mobile Support
The design is fully responsive. It works correctly on mobile phones and tablets, allowing users to scroll and interact with the table easily on small screens.

![Mobile Version](https://raw.githubusercontent.com/adfchcjnkb/periodic-table/refs/heads/main/main%20page%20(mobile)%20.png)

### 4. About the Project
The platform includes an "About" page that introduces the team and the goals of the project.

![About Page](https://raw.githubusercontent.com/adfchcjnkb/periodic-table/refs/heads/main/about%20page%20(Desktop)%20.png)

## Technical Details

- **Frontend:** Built using HTML5, CSS3, and Vanilla JavaScript.
- **Data:** Element information is stored and managed using JSON files.
- **Design:** Modern UI with rounded elements and interactive animations.
